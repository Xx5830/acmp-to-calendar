# acmp-to-calendar

Утилита для автоматического отслеживания вашего прогресса на [acmp.ru](https://acmp.ru) и синхронизации задач с вашим персональным календарем.

Больше не нужно держать в голове, какие задачи вы планировали решить к олимпиаде или собеседованию - планируйте свой тренировочный процесс прямо в привычном календаре!

---

## Основные возможности

- **Синхронизация прогресса:** Автоматически подтягивает список решенных, начатых и неразобранных задач с вашего профиля ACMP.
- **Экспорт в календарь:** Создает события для выбранных задач (дедлайны, время на решение).
- **Поддержка форматов:** Экспорт в универсальный формат `.ics` или прямая интеграция через Google Calendar API.
- **Трекинг статистики:** Помогает распределять нагрузку и не забрасывать спортивное программирование.
